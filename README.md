# PeMMO
The MMO plugin for pocketmine servers
# Features
- Mining Skill
- Foraging / Treecutting Skill
- Crafting Skill
- Building Skill
- Levelup Rewards
- Custom Config
# Comming Features
- Combat / Hunter Skill
- Explorer Skill
- Murder Skill
